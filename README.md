A clean and modern blog platform built with **Python & Django**.  
Frontend UI was crafted with AI assistance.

---

## ✨ Features

- User Registration & Login
- Create and View Blog Posts
- Markdown Content Rendering
- Nested Comment System
- Like Button (AJAX-based)
- Clean & Modern UI

---

## 🛠 Tech Stack

- **Backend:** Python, Django  
- **Frontend:** HTML, CSS, JavaScript  
- **Markdown Rendering:** markdown2  

---

## 🚀 Run Locally
```bash
git clone https://github.com/mobin-abdi/zana-blog.git
cd zana-blog
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
‍‍‍```
